# Data Dictionary

| Column Name | Description |
| :--- | :--- |
| **Name** | The full name of the user. |
| **Date Joined** | The date the account was created. |
| **City** | The city of operation (Chicago, Houston, LA, or New York). |
| **Rating** | Average star rating for the driver. |
| **Trips Made** | Total count of completed trips per driver. |
| **Total Revenue** | Total earnings generated from fares (USD). |

### Dataset Overview
- **Total Users:** 2,000 (1,600 Riders / 400 Drivers)
- **Active Driver Distribution:** 
    - New York: 98
    - Houston: 94
    - Chicago: 81
    - Los Angeles: 78
